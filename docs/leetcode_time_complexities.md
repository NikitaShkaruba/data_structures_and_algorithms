# Acceptable leetcode.com time complexities

- `n <= 10` - factorial `O(n!)` or slow exponential `O(4^n)`
- `n <= 20` - fast exponential `O(2^n)`
- `n <= 100` - cubic `O(n^3)`
- `n <= 1000` - quadratic `O(n^2)`
- `n <= 10^5` - loglinear `O(n*logn)`
- `n <= 10^6` - linear `O(n)`
- `n < âˆž` - logarithmic `O(logn)` or constant `O(1)`
